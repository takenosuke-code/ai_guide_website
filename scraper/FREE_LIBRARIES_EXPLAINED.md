## What is a "User Agent"?

The **user agent** is just a text string that identifies your scraper to websites. It's like saying "Hi, I'm Chrome browser" or "Hi, I'm a Python scraper."

### Examples:
- Chrome browser: `Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36...`
- Our scraper: `Mozilla/5.0 (compatible; AI-Scraper/1.0)`

### Why Use It?
- Some websites block requests without a user agent
- It's polite to identify yourself
- It's just a text string - **completely free, no service needed**

### Can You Change It?
Yes! You can use any text you want:
```yaml
user_agent: "My Custom Scraper v1.0"
user_agent: "Mozilla/5.0 (Windows NT 10.0; Win64; x64)"
user_agent: "AI Tool Updater Bot"
```
